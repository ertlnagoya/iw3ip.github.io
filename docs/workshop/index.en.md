# Workshop (Flow Design)

This section is for instructors/TAs to run the full workshop session.

## Workshop vs Hands-on

- Workshop: end-to-end facilitation plan
- Hands-on: practical execution recipes for participants

Flow:
1. Explain architecture and goals
2. Move participants to the selected Hands-on path
3. Return to workshop wrap-up and extension topics

## Recommended timeline (120 min)

1. Intro (10)
2. Environment check (20)
3. Quickstart boot (25)
4. Hands-on execution (40)
5. Mobile viewer verification (15)
6. Wrap-up (10)
