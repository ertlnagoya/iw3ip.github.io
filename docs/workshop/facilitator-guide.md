# 講師進行ガイド

## ゴール

受講者が以下を一連で実行できる状態にする。

1. イベントデータ生成
2. 自動商品化
3. 購入
4. データ受け取り

## 進行テンプレート

### 1. 導入（10分）

- データ共有基盤の目的
- オンチェーン/オフチェーンの役割分担

### 2. 起動（20分）

- [最短起動](quickstart.md) を全員で実施
- ここで詰まる人を優先サポート

### 3. 実習（40分）

- HUSKYLENS2班: [hands-on/huskylens2.md](../hands-on/huskylens2.md)
- Webcam班: [hands-on/webcam.md](../hands-on/webcam.md)

### 4. スマホ確認（15分）

- [hands-on/mobile-viewer.md](../hands-on/mobile-viewer.md)

### 5. まとめ（15分）

- うまくいった点/詰まった点の共有
- 発展課題提示

## 講師向けチェックリスト

- [ ] 全サービス起動済み
- [ ] MetaMaskテストアカウント準備済み
- [ ] `raw_data/output` への書き込み確認済み
- [ ] `/mobile` 接続確認済み
