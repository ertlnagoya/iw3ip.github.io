# Prerequisites

## Required

- Docker
- VSCode + Dev Containers
- Node.js / npm
- Rust / Cargo
- Git

## Recommended

- Docker Desktop (for `host.docker.internal`)
- MetaMask

## Participant checks

```bash
docker --version
npm -v
cargo --version
```

```bash
git clone --recursive https://github.com/ertlnagoya/Blockchain_IoT_Marketplace.git
cd Blockchain_IoT_Marketplace
```
