# 問い合わせ先

IW3IP に関する質問、教材利用、開発協力の相談先です。

## 相談できる内容

- サイトやハンズオン手順に関する質問
- サンプル実装やドキュメントの不明点
- ワークショップや授業利用に関する相談
- 共同研究や開発協力に関する相談

## 連絡先

- 個人ウェブサイト: <https://www.ertl.jp/~yutaka/>

## 相談のしかた

質問・相談の際は、以下の情報を添えてください。

- 見ていたページ名
- 実行した手順やコマンド
- 発生した症状やメッセージ
- 利用している環境
  - OS
  - Docker / ブラウザ / Python などのバージョン

## 開発協力について

開発協力や共同研究に関心がある場合は、関心テーマと現在の取り組み内容を添えて連絡してください。  
IoT データ共有、SSI / DID / VC、イベント共有、AI による計画生成・制御は、本サイトの内容と直接関連する領域です。
