# 問い合わせ先

このページでは、IW3IP に関する質問、教材利用、開発協力の相談先を案内します。

## 相談できる内容

- サイトやハンズオン手順に関する質問
- サンプル実装やドキュメントの不明点
- ワークショップや授業利用に関する相談
- 共同研究や開発協力に関する相談

## 連絡先

- 個人ウェブサイト: <https://www.ertl.jp/~yutaka/>

## 相談のしかた

質問や相談を送るときは、次の情報があると状況を把握しやすくなります。

- 見ていたページ名
- 実行した手順やコマンド
- 発生した症状やメッセージ
- 利用している環境
  - OS
  - Docker / ブラウザ / Python などのバージョン

## 開発協力について

開発協力や共同研究に関心がある場合は、関心のあるテーマや、現在取り組んでいる内容が分かる形で連絡してください。  
特に、IoT データ共有、SSI / DID / VC、イベント共有、AI を使った計画生成や制御に関する話題は、このサイトの内容とつながりやすい領域です。
