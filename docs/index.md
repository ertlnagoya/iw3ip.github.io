# IoTxWeb3 Intelligence Platform (IW3IP)

IoTxWeb3 Intelligence Platform (IW3IP) は、
IoT・組込みシステム研究の実践知を土台に、
データ交換からAI判断・制御連携まで段階的に統合するための基盤です。

このサイトは、研究紹介ページをベースに、以下を整理したポータルです。

- 研究概要
- 研究キーワード
- オープンソース
- 代表論文
- 学生向けメッセージ
- IW3IPのロードマップ
