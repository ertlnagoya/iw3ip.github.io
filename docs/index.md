# IoTxWeb3 Intelligence Platform (IW3IP) ドキュメント

このサイトは、ワークショップ運営と実習手順を分けて管理するためのドキュメントです。

## ドキュメントの考え方

- **Workshop**: 講師・TA向けの進行設計（目的、時間配分、運営）
- **Hands-on**: 受講者向けの実作業手順（コマンド、期待結果、確認ポイント）

つまり、**Workshop の一部として Hands-on を実施する**構成です。
