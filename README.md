# IoTxWeb3 Intelligence Platform (IW3IP) Docs Site

This repository hosts the public documentation site for
**IoTxWeb3 Intelligence Platform (IW3IP)**.

## Local preview

```bash
pip install -r docs/requirements.txt
mkdocs serve
```

Open: `http://127.0.0.1:8000`

## Publish
